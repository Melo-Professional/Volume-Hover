# Volume-Hover

![](https://img.shields.io/static/v1?label=made%20with&message=%E2%9D%A4&color=red)

Controls application audio volumes instantly by

- *hovering over the Windows system tray icon*
- *keyboard shortcut to currently active program*
- *mouse wheel to current program under mouse pointer*
